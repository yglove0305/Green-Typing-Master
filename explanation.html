초록타자를 더 알고 싶으신가요?

이 문서는 초록타자 3의 HTML 구조와 기능을 상세히 설명합니다.

개요

초록타자는 타자 실력을 키울 수 있는 웹 기반 도구로, 실시간 진행 상황, 다양한 설정 옵션, 그리고 기록 저장 및 불러오기 기능을 제공합니다.

주요 HTML 요소

  1. 다크 모드 버튼: 다크 모드와 라이트 모드를 전환하는 버튼입니다.
  2. 컨테이너: 페이지의 모든 주요 콘텐츠를 포함하는 영역입니다.
  3. 설정 섹션: 모드, 난이도, 폰트 크기를 설정할 수 있습니다.
  4. 샘플 텍스트: 사용자가 타이핑해야 할 문장을 표시합니다.
  5. 진행률 표시: 진행 상황을 시각적으로 표시합니다.
  6. 타이핑 영역: 사용자가 입력하는 공간입니다.
  7. 통계 섹션: 시간, WPM(분당 타수), 정확도, 오류 수를 표시합니다.
  8. 리더보드: 최고 기록을 표시하는 섹션입니다.
  9. 차트 섹션: 연습 기록을 시각화하여 보여줍니다.
  10. 최근 기록 섹션: 가장 최근의 타이핑 기록을 표시합니다.
  11. 진행 상황 저장 버튼: 현재 진행 상황을 저장할 수 있습니다.
  12. CSV 내보내기 버튼: 타이핑 기록을 CSV 파일로 저장합니다.
  13. 진행 상황 불러오기 버튼: 이전에 저장한 진행 상황을 불러옵니다.
  14. 타이핑 오류 소리: 타이핑 오류 발생 시 음향 피드백을 제공합니다.
  15. 차트 라이브러리: Chart.js 라이브러리를 사용하여 기록을 시각화합니다.
  16. 다크 모드 스타일링: 다크 모드 활성화 시 스타일이 변경됩니다.
  17. 로컬 저장소: 기록과 리더보드를 저장할 때 사용됩니다.
  18. 모드 선택: 무제한 모드, 시간 제한 모드, 오류 제한 모드 중 선택 가능합니다.
  19. 난이도 선택: 쉬움, 보통, 어려움 난이도 중에서 선택 가능합니다.
  20. 폰트 크기 선택: 사용자가 타이핑 영역의 폰트 크기를 조정할 수 있습니다.
  21. 진행 상황 저장 파일 형식: JSON 형식의 파일로 저장됩니다.
  22. 기록 불러오기: JSON 파일을 업로드하여 기록을 복구합니다.
  23. 비율 통계: 정확도와 오류 비율을 계산하여 표시합니다.
  24. 반응형 디자인: 다양한 화면 크기에서 잘 작동하도록 설계되었습니다.
  25. 탭 키가 방지됨: 타이핑 영역에서 탭 키를 사용할 수 없습니다.
  26. 타이핑 완료 감지: 사용자가 문장을 정확히 입력했는지 확인합니다.
  27. 기록 저장 시점: 타이핑 도전이 완료될 때 기록이 저장됩니다.
  28. 모드별 제한: 시간 제한 모드와 오류 제한 모드는 각각 다른 조건을 적용합니다.
  29. 애니메이션 초기화: 특정 UI 요소가 화면에 나타날 때 애니메이션이 발생합니다.
  30. 진행률 계산: 문장을 얼마나 완료했는지 계산하여 진행률에 반영합니다.
  31. 모바일 최적화: 모바일 사용자를 위해 특별히 설계된 요소들이 포함됩니다.
  32. 기록 시각화: 이전 기록을 차트로 보기 쉽게 제공합니다.
  33. 다크 모드 토글: 버튼을 클릭하면 다크 모드와 라이트 모드가 전환됩니다.
  34. 에러 하이라이팅: 사용자가 틀린 문자를 빨간색으로 강조합니다.
  35. 정확도 계산: 입력한 문자의 정확도를 실시간으로 계산합니다.
  36. 기록 삭제: 이전 기록을 삭제할 수 있는 옵션을 제공합니다.
  37. 사용자 정의 가능: CSS 및 JavaScript를 수정하여 사용자 맞춤 설정이 가능합니다.
  38. 실시간 통계: 타이핑 속도와 정확도를 실시간으로 업데이트합니다.
  39. 타이핑 감지: 사용자가 입력을 시작했을 때 타이머가 시작됩니다.
  40. 진행률 애니메이션: 진행률 바가 부드럽게 증가합니다.
  41. 사용자 경험 개선: 시각적 피드백과 음향 효과를 통해 사용자의 경험을 향상시킵니다.
  42. 타이핑 기록 저장: 사용자의 타이핑 기록을 로컬 저장소에 저장합니다.
  43. 샘플 문장: 다양한 샘플 문장이 포함되어 있어 사용자의 타이핑 실력을 테스트합니다.
  44. 기록 내보내기: 사용자가 데이터를 외부 파일로 내보낼 수 있습니다.
  45. 타이핑 오류 감지: 틀린 문자를 실시간으로 감지하여 강조합니다.
  46. 음향 효과: 타이핑 오류와 성공 시 음향 효과를 제공합니다.
  47. 비동기 처리: 저장 및 불러오기 기능이 비동기적으로 처리됩니다.

설명을 찾을수 없다면 검색하세요!

추가 기능

  • 진행 상황 저장: 로컬 저장소를 이용하여 진행 상황을 저장하고 복구할 수 있습니다.
  • CSV 내보내기: 타이핑 기록을 CSV 파일로 저장할 수 있습니다.
  • 차트 시각화: 연습 기록을 차트 형태로 표시합니다.
  • 음향 피드백: 오류 발생 시 소리로 알림을 제공합니다.
  • 반응형 디자인: 다양한 화면 크기에서 잘 작동하도록 설계되었습니다.
